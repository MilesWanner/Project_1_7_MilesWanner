# Project 1 - Python Automotive Quiz

## Author
Miles Wanner

## Description
This project is a command-line quiz game designed to test new automotive recruits on vehicle makes and body types. The program allows users to select a vehicle from a list and then challenges them to correctly identify its make and body style.

The goal of the project is to reinforce knowledge of common vehicles for both the user and new employees.

## Features
- Vehicle selection from a predefined dictionary
- Input validation for user selection
- Two-stage quiz (make and body type)
- Limited attempts (three strikes)
